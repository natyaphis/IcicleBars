# IcicleBars

IcicleBars is a World of Warcraft addon that displays Frost Mage Icicle stacks as configurable bars.

## Features
- Shows Icicle stacks (0–5)
- Configurable bar size and position
- Frost specialization detection
- Multi-language support (enUS / zhCN)

## Command

/ib

Open configuration panel.

## Installation
Download the repository and place the **IcicleBars** folder into:
World of Warcraft/retail/Interface/AddOns/

Reload the game.
